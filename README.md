This repository contains a collection of Exploratory Data Analysis (EDA) projects focused on understanding real-world datasets through structured analysis, visualization, and clear communication of insights.

The purpose of this portfolio is to demonstrate:

- Data cleaning and preprocessing workflows
- Exploratory and statistical analysis
- Data visualization and interpretation
- Reproducible project structures
- Analytical storytelling

---

## About

This portfolio showcases projects developed to strengthen analytical thinking and practical data analysis skills using real datasets.  
Each project follows a structured workflow including data understanding, cleaning, univariate analysis, multivariate analysis, and communication of key findings.

## Reproducibility

Clone the repository:

```bash
git clone https://github.com/N4VHI/navhi-eda-portfolio.git
```

Create the environment:

```bash
conda env create -f environment.yml
conda activate YOUR_ENVIRONMENT
```

Navigate to any project folder and open the notebooks to reproduce the analysis.
